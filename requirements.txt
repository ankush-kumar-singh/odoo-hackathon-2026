# ==============================================================================
# TransitOps Logistics Platform - Technical Requirements & Prerequisites
# ==============================================================================

# 1. RUNTIME ENGINE
# ------------------------------------------------------------------------------
Node.js: v18.0.0 or higher (Recommended: v20.x or v22.x LTS)
npm: v9.0.0 or higher (or alternative package managers: yarn / pnpm)

# 2. DATABASE ENVIRONMENT
# ------------------------------------------------------------------------------
MongoDB: v6.0 or higher
- Can be a local MongoDB Community Server instance or a MongoDB Atlas Cloud Cluster.
- Must support standard MongoDB Connection String Protocol (mongodb:// or mongodb+srv://).

# 3. ENVIRONMENT VARIABLES (.env)
# ------------------------------------------------------------------------------
# Create a .env file in the root directory with the following variables:

# MongoDB connection string
MONGODB_URI=mongodb+srv://<username>:<password>@cluster0.ttlmuoo.mongodb.net/fleet_db?appName=Cluster0

# Gemini AI Integration API Key (Optional / Required for AI Assistant features)
GEMINI_API_KEY=your_gemini_api_key_here

# The hosting URL of the application
APP_URL=http://localhost:3000

# 4. RUNTIME COMMANDS & SCRIPTS
# ------------------------------------------------------------------------------
# Install all required packages:
#   npm install
#
# Run the development environment:
#   npm run dev
#
# Build the application for production:
#   npm run build
#
# Start the production application:
#   npm run start
